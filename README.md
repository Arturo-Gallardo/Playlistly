# Playlistly

Visual canvas for YouTube playlists — pan, zoom, and arrange videos on a grid.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.local` and set:

- `YOUTUBE_API_KEY` — public playlist loads
- `AUTH_GOOGLE_ID` / `AUTH_GOOGLE_SECRET` / `NEXTAUTH_SECRET` / `NEXTAUTH_URL` — Google sign-in
